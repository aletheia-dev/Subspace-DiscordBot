                                    Changelog



Version 1.7 (08.03.2025)
- Displaing of a player stats table as a pinned message in the Discord chat
  channel.
- Automatical extension of the message blacklist for repeated arena messages.
- New command !del to delete Discord messaged that are created by the bot.
- New command !restart to force a restart of the bot.
- New configuration parameter DiscordServerType with values 'svs4v4proleague',
  'nexus', 'continuum'. The parameter value 'continuum' does not apply for
  parameter SpecialFunction anymore.

Version 1.6 (12.01.2025)
- New configuration parameter value 'continuum' for parameter SpecialFunction 
  to make the bot only send practive and match announcements to Discord.
- New configuration parameter AltTabCount to define the number of times the
  Tab key is pressed while holding Alt when switching to the Subspace chat
  window.
- New clickable game stats link in chat transfer to Discord printed after a
  stats table.
- New discord slash command /del <message Id> to delete a bot message in the
  discord chat channel.

Version 1.5 (11.08.2024)
- New startup procedure: Automatic launch of Subspace, go to the arena that 
  the bot has last been in and then activate message transmission.
- For config parameter SpecialFunction=recording, the bot does not switch to
  the Subspace chat window at startup.
- Adding of a mention @discomod behind a 'disconnected from server' message.
- Automatic recovery when Subspace loses connection to the server.

Version 1.4 (07.07.2024)
- New configuration parameter ChatChannelIds to define an arena-specific list 
  of Discord chat channel Ids in the format <arena>:<channed Id>,[..]. If the 
  bot is in one of these arenas, the related Discord chat channel is applied 
  for message transfer.

Version 1.3 (12.04.2024)
- New bot command !spam to post a practice spam message for the bot's arena 
  to the related Discord channel.
- Replacing of user mentions in messages from Discord to Subspace with the 
  related user names.
- New configuration file Operators.txt to define operator names and levels.

Version 1.2 (28.02.2024)
- Initiating a private message transfer from a Subspace player to a Discord 
  user by means of the Discobot command in the format /!pm <name>:<message>
  The related Discord user receives the private message from the player as a 
  direct message from Discobot.
- Once a direct message channel between a Discord user and Discobot has been 
  established, it can be used to send messages from Discord to the related 
  Subspace player simply by sending direct messages to the bot. 
- After a first private message has been sent to a Discord user, the related 
  player can omit the Discord user name in succeeding private messages to this 
  Discord user.
- Initiating a private message transfer from a Discord user to a Subspace 
  player by means of the Discord command in the format /pm <name> <message>

Version 1.1 (11.02.2024)
- Transmitting of public and team messages from Discord to Subspace. A team 
  message typed in Discord has to start with two slashes. A Discord user can
  only send team messages to the default spectator frequency 8025.
- Executing Discobot commands from Discord. A bot command has to start with a 
  slash. The bot sends a reply on the Discord channel that is only visible to 
  the user who issued the command.
- Formatting of group mentions in public and team messages to be applicable in
  Discord.

Version 1.0 (04.02.2024)
- Transmitting of formatted public, team and arena messages from Subspace to 
  Discord. Public and team messages are shown with dedicated icons.
- Filtering out of messages to Discord that start with a special character for
  commands.
- Textfile 'TransferBlacklist' for substrings of messages that shall not be 
  propagated to Discord, like repeatedly sent arena messages from other bots.
- Config parameter 'MentionBlacklist' for filtering out blacklisted mentions 
  from messages to Discord.
- Adding of the mention @4vs4prac to arena messages sent to Discord as replies
  to !spam commands.
